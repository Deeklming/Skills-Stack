# 컴포넌트

## 기본
```html
<!-- ./header.svelte -->
<h1>header!</h1>

<!-- ./content.svelte -->
<h1>Content!</h1>

<!-- ./footer.svelte -->
<h1>Footer!</h1>

<!-- ./App.svelte -->
<script>
	import Header from './header.svelte';
	import Content from './content.svelte';
	import Footer from './footer.svelte';
</script>

<Header />
<Content />
<Footer />
<h2> 중복가능 </h2>
<Content />
<Footer />
```
컴포넌트는 반드시 대문자로 시작해야 한다   
각 컴포넌트 하나에 많은 기능이 담긴 것 보다는 하나의 기능이 담긴 것이 좋다
