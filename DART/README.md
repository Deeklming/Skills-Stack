# DART
[DART Homepage](https://dart.dev/)
[DART Package Repository](https://pub.dev/)

## Basic Form
```dart
void main() {
  print('dart!');
}
```

## Language Version
Dart 3+

## CLI Compile and Run
```
dart dartfile.dart
```

## Comments (주석)
```dart
// var dart = 0;
/*
...
*/
/// ...
```
//: 한줄 주석   
/* */: 여러줄 주석   
///: 문서화 주석

## 표기법
변수와 함수명: camelCase   
클래스: PascalCase   
파일명: snake_case
