# 연산자

## 연산자 우선순위
```
1 . [] ()
2 expr++ expr--
3 ++expr --expr +(부호) -(부호) ~ !
4 * / %
5 +(산술) -(산술)
6 >> << >>>
7 < <= > >= instanceof
8 == !=
9 &
10 ^
11 |
12 &&
13 ||
14 b?t:f(삼항)
15 = += -= *= /= %= &= ^= != <<= >>= >>>=
```

#### bit
비트연산자 >>, <<는 부호를 신경쓰며 비트를 움직이고 >>>는 부호를 무시하며 비트를 움직인다

#### instanceof
instanceof연산자는 객체의 타입을 확인할 때 사용하고 true 또는 false를 반환한다
> 사용법: instance instanceof class
