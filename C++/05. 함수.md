# 함수

## function
```c++
#include <iostream>

using namespace std;

void fA(int i){
    cout<<"i: "<<i<<endl;
    cout<<"This function name: "<<__func__<<endl;//c++에는 함수마다 __func__로컬 변수를 가짐
}

auto addfB(int n1, int n2){//c++14 이후 가능
    return n1+n2;
}

int main(){
    fA(5);
    int b = addfB(1,2);
    cout<<"b: "<<b<<endl;
}
```
c++에서 __func__는 해당 함수의 이름을 가지고 있다
