# Bash

## 기본

리눅스의 기본 스크립트이다

```bash
#!/bin/bash

echo "bash" #기본 출력
```

`#!/bin/bash`은 셸 스트립트라는 것을 알려주는 목적이다

## 실행

터미널에서 실행

```txt
- sh 명령어로 sh파일 실행1
sh bashfile.sh

- sh파일에 실행 권한을 줘서 실행2
chmod +x bashfile.sh
./bashfile.sh

- sh파일이 아닌 프롬프트에서 바로 실행3
echo "bash"
```

## 주석

```bash
# 한 줄 주석
```

`#`으로 주석처리 한다
